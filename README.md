# Holly & Oak Yarn Co.

A simple informational website for Holly & Oak Yarn Co., a community yarn shop in South Bend’s Near Northwest neighborhood. Built with HTML, CSS, and JavaScript (no build step).

## What’s included

- **About** — Shop intro and values (fellowship, creativity, inspiration, community)
- **Visit & Contact** — Business card image, phone, email, address
- **Gallery** — Click any image to open it in a lightbox
- **Connect** — Links to Instagram, TikTok, and Facebook

## Project structure

```
├── index.html      # Single-page site
├── styles.css      # Layout and styling
├── script.js       # Nav, lightbox, back-to-top, hero CTA
├── images/         # Hero, contact card, and gallery images
└── README.md
```

## Running the site

1. **Option A:** Open `index.html` in a browser (file:// works for basic use).
2. **Option B:** From the project root, run a local server (e.g. `npx serve .`) and open the URL shown.

## Images

Place images in the `images/` folder. The site expects:

| File | Use |
|------|-----|
| `HollyOakFrontDesk.jpg` | Hero image |
| `HollyOakCard.jpg` | Contact card in Visit & Contact |
| `HollyOakInventory.jpg` | Gallery 1 |
| `HollyOakStore.jpg` | Gallery 2 |
| `HollyOakClass.jpg` | Gallery 3 |

You can reuse photos from:

- [Instagram @hollyandoakyarnco](https://www.instagram.com/hollyandoakyarnco/)
- [TikTok @hollyandoakyarnco](https://www.tiktok.com/@hollyandoakyarnco)
- [Facebook group](https://www.facebook.com/groups/1349482979379219/)

If an image is missing, the hero shows “Image unavailable” and gallery slots show placeholders.
